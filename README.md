# Simple Plugin Organizer

This is an [Obsidian](https://obsidian.md/) plugin that enables organizing both core and community plugins in one setting page. 

You can use this plugin to group and cateogorize plugins for easier management. 


## Demo 
![demo](/media/demo.gif)


## How to use 

You can activate this plugin within Obsidian by doing the following:
- Open Obsidian 
- Open Setting > Community plugins
- Turn on community plugins if you haven't 
- Browse > Search for **Simple Plugin Organizer** > Install > Enable


## Latest Release 

### 1.0 - First Release 
- Features
	- group and list all plugins in a single setting page 
	- assign colors to groups
